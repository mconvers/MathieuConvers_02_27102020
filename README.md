# MathieuConvers_02_27102020 (Projet formation)

## Install

> Installation requires
> * nodejs 
> * yarn

```
git clone https://github.com/mconvers/MathieuConvers_02_27102020.git
cd MathieuConvers_02_27102020
yarn install
```

## Develop

```
yarn run dev
```

## Build

```
yarn build
```
